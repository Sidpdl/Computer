Pre-processor Directives

Pre-processor directives give instructions to the compiler before actual compilation. They manage macros, file inclusion, and conditional compilation.

  • Macros
  • File Inclusion
  • Conditional Compilation
  • Other directives
#define MAX 100 #define SQUARE(x) ((x)*(x)) #include <stdio.h> #ifdef DEBUG #define LOG(msg) printf(msg) #else #define LOG(msg) #endif

Types of Tokens

Tokens are the smallest elements in C language code such as keywords, identifiers, constants, operators, and symbols. They form the building blocks of programs.

  • Keywords
  • Identifiers
  • Constants
  • Strings
  • Special Symbols
  • Operators

Input/Output Functions

C provides formatted and unformatted input/output functions to interact with users, such as printf, scanf, getchar, and puts.

  • Formatted I/O Functions (printf(), scanf())
  • Unformatted I/O Functions (putchar(), getchar(), puts(), gets())
#include <stdio.h> int main() { char name[50]; int age; printf("Enter name: "); gets(name); printf("Enter age: "); scanf("%d", &age); printf("Hello, %s!\n", name); printf("You are %d years old.\n", age); return 0; }

Control Statements

Control statements guide program flow using condition checks or jumps, including switch-case and goto labels for decision making.

  • Switch case
  • Goto label
int main() { int choice = 2; switch(choice) { case 1: printf("One"); break; case 2: printf("Two"); break; default: goto error; } return 0; error: printf("Invalid choice"); return 1; }

Arrays

Arrays store collections of elements of the same type in contiguous memory locations. They support operations like traversal and transposition.

  • One Dimensional Array
  • Two Dimensional Array
  • Array Operations (1D and 2D): +, -, *, /, transpose
int main() { int arr1D[5] = {1, 2, 3, 4, 5}; int matrix[2][2] = {{1, 2}, {3, 4}}; int transpose[2][2]; // Transpose for(int i = 0; i < 2; i++) for(int j = 0; j < 2; j++) transpose[j][i] = matrix[i][j]; }

Patterns

Patterns are sequences of characters printed in specific shapes using loops. Common patterns include pyramids, squares, and rectangles.

  • Square
  • Rectangle
  • Right Pyramid
  • Inverse Right Pyramid
void printPyramid(int n) { for(int i = 0; i < n; i++) { for(int j = 0; j <= i; j++) { printf("* "); } printf("\n"); } }

String Functions

String functions manipulate character arrays by measuring length, copying, concatenating, and comparing strings efficiently.

  • strlen()
  • strlwr()
  • strupr()
  • strcpy()
  • strcmp()
  • strcat()
  • strncat()
